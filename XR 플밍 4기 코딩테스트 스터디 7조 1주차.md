### 문제 1 짝수의 합
https://school.programmers.co.kr/learn/courses/30/lessons/120831

- 김세진 : i를 2로 나눈 값을 모두 더하는 함수를 제작했다.
- 김영찬 : 상동, 수학공식 써서 푼 사람도 있어서 대단하다고 생각했다.
- 나현수 : i를 0부터 시작하고 i+2해서 해결, i<=n
- 이상현 : i = 2로 하고 나머지는 상동
- 문조영 : 삼항연산자를 사용하여 if문 안쓰고 답안 작성

- 제안 - 이상현 : 깃허브 레포스토리를 파서 공유해보는거 어떨지?

### 문제 2 삼각형의 완성 조건
https://school.programmers.co.kr/learn/courses/30/lessons/120889

- 나현수 : 숫자를 담은 크기순으로 배열정렬 해서 [2]와 [0]+[1]을 비교해서 출력
  - 확장 매서드 linq 사용
- 김영찬 : 숫자 3개를 다 반환해서 값을 계산
- 이상현 : 가장 긴 변 을 구하고 전체의 값을 구해서 가장 긴 변의 값을 빼서 계산
  - if 대신 삼항연산자 사용
  - 문법적으로 하라 : 확장매서드/삼항연산자 사용해봄
- 김세진 : 직각삼각형을 찾는 문제인줄 알고 피타고라스의 정리를 초반에 사용
  - 출제자의 의도를 잘 생각해보자
  - for문 하나만써보려고 했다 > [0]이 제일 큰수가 아니면 터졌다.
  - 선분3개 중에 동일한 선분이 2개 이상 있고 그 값이 제일 큰 값이면 간단하게 적을때 오류가 발생할 가능성이 있으니 여러 경우의 수를 if와 else if를 사용해서 모든 경우의 수를 작성했다.
- 문조영 : 처음 시도 : 3개의 숫자를 다 넣을까? > 하드코딩
  - for을 2개만 사용해보자 > 찾다가 countinue를 사용하면 그 뒤를 실행 안하기 때문에 코드를 아낄 수 있다.
  - 삼항연산자로 return 반환 : 2가지 답만 나오면 삼항연산자를 쓰면 간단하게 작성된다.
  - 일단 int answer = [0]으로 지정해두고 for문으로 찾을때는 [1]부터 찾게 했다.

- 이상현 : 선정 기준은?

### 문제 3 정수를 입력 받아 n의 약수의 합을 구하라
https://school.programmers.co.kr/learn/courses/30/lessons/12928

- 나현수 : 1번문제와 비슷하게 풀었다.
  - 1~n까지 for문으로 돌리고 n % i == 0일때를 모두 더해서 구함
- 김세진 : 상동, 약수를 모두 구해야 되나 하고 생각하다가 1부터 나눠서 떨어지는 수를 더하면 되지 않을까 해서 똑같이 풀었다.
- 문조영 / 이상현 : 상동
- 김영찬 : List로 한 숫자의 모든 약수를 구해 저장한 동적배열을 만들고 인덱스 안의 값을 foreach문으로 모두 합했다.
